# GOLDEN BAKERY

A modern and responsive bakery business website built using React.js and Tailwind CSS.

## 🌟 Features

* Responsive modern bakery UI
* Interactive bakery menu
* Add to Cart functionality
* WhatsApp ordering system
* Birthday cake order form
* Customer contact form
* Smooth scrolling navigation
* Framer Motion animations
* Fully deployed on Vercel

## 🛠 Tech Stack

* React.js
* Tailwind CSS
* Vite
* Framer Motion
* React Icons
